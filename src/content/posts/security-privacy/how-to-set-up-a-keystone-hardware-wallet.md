## Introduction to Keystone Wallet Setup

Keystone hardware wallets provide premium air-gapped security with a large touchscreen and advanced features like multisig and QR code signing. The Keystone Pro remains a top choice in 2026 for users wanting complete isolation and intuitive interface.

This guide covers full setup for the Keystone Pro (process similar for Essential model), including seed generation, backup, app pairing, and transaction signing.

## Why Keystone

Highlights:
- Fully air-gapped QR code workflow
- 4-inch touchscreen for clear verification
- Open-source firmware
- Native multisig support
- Durable build with fingerprint option (Pro)

## What You Need

- Keystone device (official purchase only)
- Smartphone or computer
- Keystone app or compatible software (MetaMask, Electrum, etc.)

## Step-by-Step Keystone Setup

### Step 1: Unbox and Verify

Check for tamper-evident packaging. Verify device authenticity via Keystone website using serial number.

### Step 2: Power On and Initial Setup

Press power button. Select language and follow onboarding.

### Step 3: Create New Wallet

Choose "Create New Wallet".  
Select 12/18/24 words and entropy strength.

Device generates seed phrase securely on-chip.

### Step 4: Backup Seed Phrase

Keystone displays words individually.  
Write them down accurately on provided backup sheet.  
Store offline in secure location.

Confirm by re-entering select words on touchscreen.

### Step 5: Set Device PIN

Create strong 6–12 digit PIN.  
Optionally enable fingerprint (Pro model).

### Step 6: Update Firmware (Recommended)

In app or companion software, check for updates. Keystone uses air-gapped QR update process.

### Step 7: Pair with Software Wallet

Options:
- Keystone app (simple)
- MetaMask (EVM chains)
- Electrum (Bitcoin)
- Sparrow Wallet (advanced)

Example with MetaMask:
- Open MetaMask mobile/desktop
- Add hardware wallet > Scan QR
- Keystone displays connection QR
- Scan to pair

### Step 8: Test Receiving

Generate receive address in paired app.  
Verify full address on Keystone screen before sharing.

Send small test transaction.

### Step 9: Signing Transactions

To send:
- Create transaction in app
- App shows unsigned QR
- Scan with Keystone camera
- Review all details on large screen (amount, address, fees)
- Confirm with PIN/fingerprint
- Signed QR appears
- Scan back to app to broadcast

## Advanced Features

## Best Practices

- Always verify full addresses and amounts on device screen
- Store seed phrase metal-backed for longevity

## Troubleshooting

- QR issues: Adjust distance/lighting
- Pairing fails: Restart both devices
- Forgotten PIN: Reset with seed phrase
