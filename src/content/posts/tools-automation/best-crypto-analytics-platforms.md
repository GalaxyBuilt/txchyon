## How to Use Analytics Platforms Safely

Analytics platforms are **view-only** — but mistakes happen if you:
- Link sensitive wallets unnecessarily
- Share private keys or seed phrases
- Ignore platform security best practices

Recommended:
- Use **read-only wallets** for analysis
- Enable **2FA** on all accounts
- Combine analytics with alert systems for faster response
